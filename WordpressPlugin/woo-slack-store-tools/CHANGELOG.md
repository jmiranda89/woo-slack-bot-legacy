# Changelog

All notable changes to this plugin are documented in this file.

## 2.0.0 - 2026-02-11
- Consolidated theme `functions.php` customizations into plugin.
- Added Woo mail log REST API endpoints.
- Added Woo admin order search extension for custom order number meta.
- Added new order email details for both web and custom order IDs.
- Added user meta registration (`customer_code`, `customer_class`) with REST support.
- Added `readme.txt` and packaging script for reuse across stores.
